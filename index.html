Boids Algorithm

Boids algorithm was developed by Craig Reynolds. More information can be found HERE.

I saw a YouTube video about this algorithm and decided to play around with it.

I have made multiple version of this algorithm, giving different user controls and changing parts of the code to allow the algorithm to work in different ways. Below, is a contents table linking to the different versions of the algorithm and what the key difference is between the versions


Algorithm Version Details
Version 1 This is my initial version of the Boids algorithm. This code is fully commented which explains some of the theories used in the algorithm. However, this version does have noticable clumping which indicated that the seperation of Boids wasn't as effective as needed.
Version 2 This version is not commented. In this version, the three forces have been weighted, with seperation being given a higher weight than the other two forces. This results in "flocking" that seems more natural and doesn't allow for clumping as much as Version 1 did.
Version 3 This version has changed code commented. In this version, the user can control each of the three forces that act upon each Boid.
