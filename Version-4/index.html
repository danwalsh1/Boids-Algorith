Boids Algorithm

Version 4

This version of the algorithm allows you to control not only the three forces that act upon each Boid, but also the perception radius of each force. You can also control the maximum speed of each Boid and the maximum force that can be applied to each Boid. The size of the world (canvas) can also be changed now.

This version of the algorithm has had the changed code commented.
